YouTube PL

Hands-Free Use

YouTube PL is literally the essence of "plug-n-play". The user simply needs to start the application and YouTube PL takes care of all the heavy lifting in the background. The goal for this project was to make it as easy as possible for the user to use. Because YouTube PL targets showrooms, it must be able to operate without user input.

Easy Install

The YouTube PL application was designed to be portable. It was created with standard Linux shell commands, which means that any Linux computer can run this program. As a result it is very usable in practical situations where a Linux machine would already be available.

Syncronizes to Playlists

YouTube PL automatically downloads the latest videos added into the playlist. You'll always have the most recent version of the playlist on YouTube. The application is adaptable. You can track various playlists and automatically download the newest added videos.
