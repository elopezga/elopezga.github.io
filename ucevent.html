Easy User Integration

UC Events was designed from the start to make it easy for user to integrate their existing digital profile (facebook) into our application. Because ur application relies on unique user identification, my team and I decided not to include our own sign up form to make our application more accessable since our.

Easy of Use

The application was designed from the ground up to allow users to quickly and easily discover events in their area. The category system categorizes each event that was submitted by users so that users wanting to browse events can choose from a category that they would like the event to be. Multiple categories can be choosen to act as a filter for events in the sytem. This allows quick and easy event browsing for users.

User Retention Design

In order to enable our userbase to thrive, we added a feature that allows for users to keep track of the events that they are going to. This is listed in the "My Events" tap. Here, a user is able to check detailed information about the events that they are subscribed to. We want users to be able to recognize the events they subscribed to; not to recall.
