Error Rate

Real-Time Data Analysis

The Error Rate library was designed to analyze error rate data as it comes in from the live network system that is being probed. This allows data to be available as soon as possible for the user. With this responsive design, the user can take correction measures to correct the system based on the data outputed by the library.

Flexability

The Error Rate library uses native Python libraries that come with the Python package. This means that Error Rate works right out of the box; no need to download or install external packages.

Integratable Design

Since the Error Rate outputs to the standard output stream, the output can be easily redirected to other devices. This means that users can build applications on top of this librar. For instance, users can integrate graphical user interfaces so that it is more interactable with users.
