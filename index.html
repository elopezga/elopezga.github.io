UC Events

UC Event is an Android application developed by Unify in San Diego. It connects users to events by allowing them to post events in a public forum. At the same time, users can browse events and show interest by toggling the "Going" button.

Error Rate
Error Rate library is a Python library for detecting and calculating bit and packet error rate of networks in real-time using wireshark. Error Rate library can be integrated in any projects for analyzing reliability of network traffic.
